# Login Seguro con Python 🔐

Sistema de autenticación básico desarrollado en Python.

## ¿Qué hace?
- Solicita nombre de usuario y contraseña de forma oculta
- Permite 3 intentos antes de bloquear la cuenta
- Usa la librería `getpass` para ocultar la contraseña al escribirla

## Tecnologías
- Python 3
- getpass

## ¿Cómo ejecutarlo?
1. Clona el repositorio
2. Ejecuta `python login.py`
